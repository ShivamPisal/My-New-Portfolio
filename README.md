# My New Portfolio

A modern animated React portfolio showcasing my projects, skills, experience, resume, and contact information.

## Overview

This portfolio is built as a premium single-page developer portfolio with smooth animations, responsive layouts, dark/light theme support, project previews, and a polished contact experience.

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- React Icons

## Features

- Responsive single-page portfolio design
- Premium animated UI and section reveals
- Dark and light theme toggle
- Smooth navbar scrolling
- Custom animated cursor on desktop
- Featured project spotlight
- Project showcase with demo previews
- Skills grouped by category
- Experience, education, and certification timelines
- Resume preview and download
- Contact form with direct social links

## Project Structure

```text
src/
  assets/          Project images, profile photo, resume, and static assets
  components/      Reusable UI components
  data/            Portfolio content and project data
  pages/           Main portfolio page
  utils/           Helper utilities
```

## Getting Started

Clone the repository and install dependencies.

```bash
npm install
```

Start the development server.

```bash
npm run dev
```

## Build

Create a production build.

```bash
npm run build
```

Preview the production build locally.

```bash
npm run preview
```

## Deployment

This project is ready to deploy on Vercel.

Recommended Vercel settings:

```text
Framework Preset: Vite
Install Command: npm install
Build Command: npm run build
Output Directory: dist
```

## Customization

Most portfolio content is stored in:

```text
src/data/portfolioData.js
```

Update this file to modify navigation links, experience, education, certifications, skills, and project details.
